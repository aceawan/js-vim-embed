js-vim-embed is an embeddable implementation of js-vim

To get started, the API is almost as simple as that of the ace editor:

<pre id='vim'></pre>
<script src='/vim.js'><script>
<script>
	vim.edit({
		el: document.getElementById('vim')
	});
<script>

js-vim itself a fairly robust implementation of vim:

	- motions
		- h, j, k, l
		- {n}f{m}
		- /foo
